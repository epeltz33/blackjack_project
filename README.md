
# Blackjack

A browser-based Blackjack game built with vanilla JavaScript. Place bets, hit or stand, and play against the dealer.

## How to run

Open `index.html` in a browser, or serve the project locally (e.g. `npx serve .` or your editor’s Live Server) so assets load correctly.

## Tech

- HTML, CSS, JavaScript (no build step)
- Bootstrap 4 & jQuery via CDN
- Card assets in `src/images/`
